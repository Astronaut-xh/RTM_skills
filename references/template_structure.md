# RTM模板详细结构

## 模板文件：RTM_AI探索_原版.xlsx

### Sheet 1: DR-FL（需求到Feature分解）

#### 表头结构
| 列 | 字段名 | 说明 |
|----|--------|------|
| A | DR编号 | 需求编号，如 DR.APLC.001 |
| B | Feature类别 | 时钟、复位、寄存器、工作模式等 |
| C | FL编号 | FL_xxx_yyy格式 |
| D | Feature描述 | 详细特性描述 |
| E | TP编号 | 关联的TP编号（由DV反标） |

#### Feature类别列表（按编号顺序）
| 编号 | 类别 |
|------|------|
| 001 | 时钟 |
| 002 | 复位 |
| 003 | 寄存器 |
| 004 | 工作模式 |
| 005 | 数据接口 |
| 006 | 控制接口 |
| 007 | 配置接口 |
| 008 | 中断 |
| 009 | 异常 |
| 010 | 低功耗 |
| 011 | 性能 |
| 012 | DFX |
| 013 | Memory map |
| 014 | 总线接口 |

#### 填写说明区域（Row 22-27）
- Row 22: "填写说明"标题
- Row 23-27: 各列的填写说明

---

### Sheet 2: FL-TP（Feature到Test Point分解）

#### 表头结构
| 列 | 字段名 | 说明 |
|----|--------|------|
| A | Feature编号 | 对应FL编号 |
| B | TP类别 | 与Feature类别类似 |
| C | TP编号 | TP_xxx格式 |
| D | TP描述 | 条件+激励+期望 |
| E | checker编号 | CHK_xxx格式 |
| F | Testcase编号 | DV_TC_xxx格式 |

#### 填写说明区域（Row 23-28）
- Row 23: "填写说明"标题
- Row 24-28: 各列的填写说明

---

### Sheet 3: Checker List（检查点清单）

#### 表头结构
| 列 | 字段名 | 说明 |
|----|--------|------|
| A | CHK编号 | CHK_xxx格式 |
| B | CHK Name | 检查点名称 |
| C | CHK描述 | 定性+定量描述 |
| D | 备注 | 附加说明 |

---

### Sheet 4: DV Testcase List（测试用例清单）

#### 表头结构
| 列 | 字段名 | 说明 |
|----|--------|------|
| A | TC编号 | TC_xxx格式 |
| B | TC Name | 测试用例名称 |
| C | TC描述 | 配置条件+输入激励+期望结果+coverage check点 |
| D | 备注 | 附加说明 |

---
